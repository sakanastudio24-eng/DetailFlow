# Cruz N Clean V2

Documentation-first rebuild of a car detailing portfolio and booking website.

## Stack
- Frontend: Next.js + TypeScript + React + Tailwind CSS + shadcn/ui (+ selective MUI)
- Backend: FastAPI (minimal API)
- Storage (V0): JSON files (designed for later Supabase migration)

## Delivery Rules
- Mobile-first UI implementation
- Commit per section
- Every function in TypeScript files must include a short documentation block
- Build one page, then stop and review

## Planned Sections
1. Routes and architecture docs
2. Project structure and scaffolding
3. First page implementation (`/` Home)

## Local Development (target)
- Web: `apps/web`
- API: `apps/api`

See `DEV_GUIDE.md` and `/docs` for workflow details.
